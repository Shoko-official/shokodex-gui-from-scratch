# Project Citations & Sources

This document lists the documentation, libraries, and external resources referenced in the **Shokodex (codex-gui-local)** project.

## Reference Documentation
* **Anyhow Crate** : Official error-handling documentation: [https://docs.rs/anyhow/latest/anyhow/](https://docs.rs/anyhow/latest/anyhow/)

## Crate Dependencies
* **anyhow** : Error handling and context propagation. [Docs](https://docs.rs/anyhow/)
* **reqwest** : Async HTTP client for LM Studio API calls. [Docs](https://docs.rs/reqwest/)
* **serde** : JSON serialization and deserialization. [Docs](https://docs.rs/serde/)
* **tokio** : Asynchronous runtime environment. [Docs](https://docs.rs/tokio/)

## Technical Inspirations
* **Ratatui** : Layout ideas for text card borders (`with_border` structure). [Docs](https://docs.rs/ratatui/latest/ratatui/)
* **Unicode-Width** : Display width calculations for Unicode characters. [Docs](https://docs.rs/unicode-width/latest/unicode_width/)
